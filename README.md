Retail Management System is a simple management system designed to handle customers and employees, built using PHP, HTML, CSS, and MySQL. 
It allows users to add customer details, place tailoring orders, search records, and view combined information from both customer and order tables.
This report presents a structured analysis of TailorPro, including its database architecture, ERD, system workflow, SQL schema, sample queries, system interface, and future development prospects. 
The system aims to enhance operational efficiency, accuracy, and customer satisfaction by providing organised and reliable data management.
